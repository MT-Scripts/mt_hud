# MT HUD V3
Simple and clean FiveM HUD made in Typescript with React and Mantine

# Features
- Player HUD:
    - Voice
    - Health
    - Armor
    - Hunger
    - Thirst
    - Oxygen/Stamina
- Vehicle HUD:
    - Speed
    - Gears
    - Compass
- Easy colors change via config file
- New minimap shape
- Clear and optimized UI

# Dependicies
- qb-core or qbx_core
- ox_lib
